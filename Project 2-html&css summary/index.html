HTML&CSS Basics Summary

HTML Summary

HTML(HyperTextMarkupLanguage) is used to define our page,content and meaning.
You don't use it for styling purposes. Use CSS for that instead!

  • HTML uses "elements" to describe(annotate) content.
  • HTML elements typpicaly have an opening tag, content and then a closing tag.
  • You can also have void(empty) elements like images.
  • You can also configure elements with atributes.
  • There's a long list of available elementsbut you'll gain experience over time, no worries.
  • Learn more about about all available HTML elements on the MDN CSS property reference.

CSS Summary

  • CSS(Cascading Style Sheets) is used for styling your page content.
  • Styles are assigned styles via property-value pairs.
  • You can assign styles via the "style" attribute.
  • To avoid code duplication, you typpicaly use global styles (e.g. via the "style"element)instead.
  • Alternatively, you can work with external stylesheet files which you "link" to.
  • When working with CSS,concepts like "inheritence","specificity","box model" should be understood.
  • Learn more about all available CSS properties and values on the MDN CSS property reference.
